# Uso de Inteligência Artificial

A IA foi utilizada como apoio para organizar ideias, revisar a estrutura do projeto, criar dados simulados coerentes, sugerir regras lógicas, melhorar comentários do código e montar a documentação.

Validação crítica feita pela equipe:
- Conferir se todos os requisitos do enunciado foram contemplados.
- Executar o código Python e verificar se roda sem erros.
- Validar se os dados simulados fazem sentido para uma missão espacial experimental.
- Ajustar nomes, RMs, links e apresentação final antes da entrega.

A equipe deve compreender, explicar e defender todas as decisões técnicas implementadas.
