# Roteiro sugerido para vídeo de até 4 minutos

1. Apresentar o problema: monitorar uma missão espacial experimental.
2. Mostrar a pasta do projeto e os arquivos obrigatórios.
3. Abrir `data/dados.csv` e explicar as variáveis.
4. Executar `python src/sistema.py` no terminal.
5. Explicar o diagnóstico, alertas, previsão e recomendações.
6. Defender as estruturas de dados: listas, fila, pilha, dicionário, hierarquia e matriz.
7. Finalizar mostrando que o README e o relatório estão completos.
